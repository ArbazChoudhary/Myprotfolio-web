DevOps Project: Setup CI/CD Pipeline using Jenkins

Duration: Aug 2024 - Sep 2024

In this project, I implemented a CI/CD pipeline to automate the deployment of a Django To-Do app hosted on an EC2 instance. The pipeline enhanced the development workflow, enabling smooth builds, testing, and deployments.

Objectives:

  • Automate Deployment: Set up a CI/CD pipeline for seamless deployment.
  • Ensure Portability: Use Docker to containerize the Django app.
  • Improve Workflow: Implement Jenkins for efficient integration and deployment.

Key Steps:

  • Provisioning: Deployed an EC2 instance and installed Jenkins for creating the CI/CD environment.
  • Pipeline Configuration: Configured Jenkins to automate code retrieval from GitHub, build, and deploy.
  • Docker Integration: Dockerized the Django app by creating a Dockerfile, ensuring the app runs consistently in various environments.
  • Deployment Automation: Developed scripts to automate the deployment process, integrating them into the Jenkins pipeline for smooth updates.
  • Testing: Integrated automated tests in Jenkins to verify functionality before deployment.

Challenges & Solutions:

  • Jenkins Setup: Resolved plugin and configuration issues with community resources.
  • Docker Compatibility: Adjusted the Dockerfile to ensure compatibility with the EC2 environment.

Outcome:

  • Streamlined Process: The CI/CD pipeline reduced deployment errors and manual intervention.
  • Efficiency: Automated testing and containerization improved consistency and speed in application updates.

Technologies Used:

  • Jenkins: CI/CD automation.
  • EC2: Application hosting.
  • Docker: Application containerization.
  • GitHub: Version control.

This project showcases my ability to implement and manage CI/CD pipelines using modern DevOps tools.
